# 통합 대시보드

Farm-Sync는 `/` 통합 대시보드로 진입한다.

## 경로 정책

- `/`: 농장 위험도, 기상, 병해충 검색, 농기계 추천, AI 리포트 진입점을 제공한다.
- `/farmer`: 기존 북마크 호환을 위해 `/`로 리다이렉트한다.
- `/admin` 및 `/admin/rentals`: 지역 운영 화면 삭제에 따라 `/`로 리다이렉트한다.
- 농장 등록 완료, 취소, 병해충 사진 검색 복귀 경로는 `/`를 사용한다.

## 사용자 인터페이스

- 데스크톱과 모바일 모두 `홈`, `농장 등록`, `알림`의 동일한 정보구조를 사용한다.
- 메뉴는 SPA 전환 대기 없이 브라우저 기본 탐색으로 대상 경로를 즉시 연다.
- 클릭 가능한 항목은 키보드 포커스 표시와 최소 44px 높이를 유지한다.
## 분석 API 호출 캐시 정책

- 대시보드 메뉴 전환은 전체 문서 reload가 아닌 SPA 전환을 사용해 QueryClient와 Zustand 상태를 유지한다.
- 농장 분석 결과는 `sessionStorage`에 농장 ID별로 저장하며, 성공한 API 응답만 캐시한다.
- 캐시 TTL은 5분이다. 5분 이내에 다른 페이지로 이동했다가 `/`로 돌아오면 저장된 분석 결과를 표시하고 분석 API를 다시 호출하지 않는다.
- API를 다시 호출하는 기준은 새로고침 버튼 클릭, 분석 농장 변경, 캐시 TTL 만료, 캐시 없음, 직전 API 실패다.
- 분석 API 호출이 실패하면 해당 농장의 분석 캐시를 삭제해 다음 진입 시 재시도한다.
- 농장 목록도 5분 TTL의 Zustand 메모리 캐시를 사용한다. Supabase realtime 변경 이벤트는 이 캐시를 우회해 즉시 재조회한다.
